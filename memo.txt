##Command line##

==まとめ==
touch beginner.txt ファイルを作成する
cat beginner.txt ファイルの中身を表示する
mkdir html ディレクトリを作成する　make directory
cd html ディレクトリを移動する　change directory
pwd 作業中のディレクトリを確認する　print working directory
ls ディレクトリの中身を確認する　list
cd .. 1つ親のディレクトリに移動する
cd ホームディレクトリに移動する
mv beginner.txt html ファイルをディレクトリに移動する　move
mv html languages ディレクトリをディレクトリに移動する
mv beginner.txt dojo.txt ファイル名を変更する
mv html about ディレクトリ名を変更する
cp dojo.txt project.txt ファイルをコピーする　copy
cp -r html ruby ディレクトリごとコピーする
rm project.txt ファイルを削除する　remove
rm -r html ディレクトリを削除する


ルートディレクトリ　1番親のディレクトリ　/で示される
過去に実行したコマンドは↑で指定できる





##git##

==手順==
0.git init gitの立ち上げ　initialize
1.コードを準備する
2.共有する準備をする
  git add ファイル名
  git commit -m "コメント"
3.共有する
  リモートリポジトリの準備(リポジトリのURLを用意する)
  リモートリポジトリを登録する際には名前をつける必要があり、一般的には「origin」とする事が多い
  git remote add リモートリポジトリ名 URL
  git push リモートリポジトリ名 master
4.ダウンロードする
  git pull リモートリポジトリ名 master
1
==まとめ==
add--commit--push--pull

git add ファイル名
git commit -m "コメント"
git remote add リモートリポジトリ名 URL （初回だけ）
git push リモートリポジトリ名 master
git pull リモートリポジトリ名 master

==その他==
git status 変更したファイルの把握、addしたファイルの確認
git diff 変更した内容まで確認する
git log コミットメッセージを確認する
git log -p コミットメッセージと変更内容まで確認する　Qを押すと終了する
