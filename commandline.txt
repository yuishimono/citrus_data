Command line

touch beginner.txt ファイルを作成する
cat beginner.txt ファイルの中身を表示する
mkdir html ディレクトリを作成する　make directory
cd html ディレクトリを移動する　change directory
pwd 作業中のディレクトリを確認する　print working directory
ls ディレクトリの中身を確認する　list
cd .. 1つ親のディレクトリに移動する
cd ホームディレクトリに移動する
mv beginner.txt html ファイルをディレクトリに移動する　move
mv html languages ディレクトリをディレクトリに移動する
mv beginner.txt dojo.txt ファイル名を変更する
mv html about ディレクトリ名を変更する
cp dojo.txt project.txt ファイルをコピーする　copy
cp -r html ruby ディレクトリごとコピーする
rm project.txt ファイルを削除する　remove
rm -r html ディレクトリを削除する


ルートディレクトリ　1番親のディレクトリ　/で示される
過去に実行したコマンドは↑で指定できる
